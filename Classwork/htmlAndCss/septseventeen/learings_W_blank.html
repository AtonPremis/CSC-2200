1. Position Relative

Question 1a: Position relative moves an element from where they would have been placed but keeps the element reserved in the document flow.

2. Position Absolute

Question 2a: Position absolute positions an element relative to its nearest relative ancestor.

Question 2b: To make a div a "positioned parent" for absolute children, you need to add position: relative to that div's CSS.

Question 2c: If you forget to add positioning to the parent container, an absolutely positioned element will position relative to the body tag instead.

Scenario: You want to put a close "X" button in the top-right corner of a modal dialog.

Question 2d: You would add position: relative to the modal container and position: absolute to the X button.

3. Position Fixed

Question 3a: Position fixed positions an element relative to the body and the element stays in the same spot when you scroll.

Question 3b: When you use position fixed on a navigation bar, you often need to add FILL_IN_BLANK_14 to the main content to prevent it from being hidden behind the fixed nav.

4. Display: Inline vs Block vs Inline-Block

Question 4a: Inline elements ignore width and height properties, while inline-block elements respect them.

Question 4b: If you want an element to flow inline with text but still respect width and height, you should use inline-block.

Question 4c: block elements create line breaks before and after them, while inline elements flow with the text.

5. Float

Question 5a: Float removes an element from the document flow and allows other content to wrap around it.

Question 5b: The main problem with floated elements is that their containers might not adjust properly around them.

Question 5c: To fix container collapse with floated children, you can add overflow: auto to the parent container.

Scenario: You want text to wrap around an image on the left side.

Question 5d: You would add float: left to the image CSS.

6. When to Use Each Method

Question 6a: To create a sticky navigation bar that stays at the top when scrolling, use position: fixed.

Question 6b: To put a close button in the corner of a dialog box, use top: 0 and right: 0.

Question 6c: To make small adjustments to an element's position without affecting other elements, use position: relative.

Question 6d: To make images and text flow together like in a newspaper article, use float.
